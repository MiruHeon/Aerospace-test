# SatelliteOrbitSimulation (위성궤도 시뮬레이션)

### - 주요 기능
위성궤도 시각화

### - 사용한 기본 파라미터
지구 반지름: R = 6371km
위성 고도: h = 300km
궤도 반경: r = R + h = 6671km
위성 공전속도: v=√(GM / r) = 7.73 km/s
공전 주기: T = 2πr/v = 약 5444초(90.7분)
궤도 경사각: 60°
각속도: w = 2π / T= 0.001154 rad/s

### - 위성 위치 계산 알고리즘
지구 중심 기준(ECEF 좌표계)에서 인공위성의 궤도가 하나의 평면에 놓여 있다고 볼때, 타원 궤도는 케플러 방정식을 풀어야 하고
근지점/원지점 속도 변화까지 고려해야 하므로 타원 궤도 대신 원형 궤도로 가정한다고 했습니다.
그렇게 된다면 위성의 속도와 반지름이 일정하므로 시간 t에서의 위성 진각 θ(t)는 다음과 같이 정의됩니다. θ(t) = 2πt / T

따라서 최종 위치 계산 코드는
(x,y,z)=(r cos θ,r sin θ cos 𝑖,r sin θ sin 𝑖) 로 설계되었고, 이를 10.0초 간격으로 x,y,z의 위치를 'orbit.txt'에 저장시킨 후
파이썬에서 'orbit.txt' 파일을 불러와 'matplotlib' 라이브러리를 사용하여서 시각화를 구현하였습니다.
